C#

This guide will help you get started with building applications with LUSID by FINBOURNE and the C# programming language.

FINBOURNE develops and maintains two versions of the LUSIDÂ® C# SDK:

lusid-sdk-csharp

supports Production and Early Access API endpoints

lusid-sdk-csharp-preview

supports Production, Early Access, Beta and Experimental API endpoints.

For more details on API endpoint categories, see What is the LUSID feature release lifecycle? To find out which category an API endpoint falls into, see LUSID API Documentation.

You should only use lusid-sdk-csharp-preview if you want to use the Beta or Experimental API endpoints.

Installation

The SDK is available on Nuget and can be installed by running the following:

dotnet add package Lusid.Sdk

The preview version of the SDK can be installed by running the following:

dotnet add package Lusid.Sdk.Preview

Configuring API Client

Now that you’ve installed the SDK, it’s time to use it. First, import the following modules:

dotnet new console -o FinbourneCSharpDemo
cd FinbourneCSharpDemo
using Lusid.Sdk.Api;
using Lusid.Sdk.Model;
using Lusid.Sdk.Utilities;

You can create the API factory by running the following code:

var secretsFile = "/path/to/secrets.json";
var apiFactory = LusidApiFactoryBuilder.Build(secretsFile);

You will need to create a secrets file containing your LUSID credentials. For instructions on how to construct this file, see Configure secrets file.

Now that the API client is ready, you are ready to use the various API endpoints. An API endpoint can be constructed by calling apiFactory.Api(Lusid.Sdk.Api.<className>) for any of the returned classes. For example, to construct the Transactions Portfolio API, run the following:

var txPortoliosApi = apiFactory.Api<ITransactionPortfoliosApi>();
