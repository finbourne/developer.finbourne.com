Python

This guide will help you get started with building applications with LUSID by FINBOURNE and the Python programming language.

FINBOURNE develops and maintains two versions of the LUSIDÂ® Python SDK:

lusid-sdk-python

supports Production and Early Access API endpoints

lusid-sdk-python-preview

supports Production, Early Access, Beta and Experimental API endpoints.

For more details on API endpoint categories, see What is the LUSID feature release lifecycle? To find out which category an API endpoint falls into, see LUSID API Documentation.

You should only use lusid-sdk-python-prevew if you want to use the Beta or Experimental API endpoints.

Installation

The SDK is available on PyPi and can be installed by running the following:

pip install lusid-sdk

The preview version of the SDK can be installed by running the following:

pip install lusid-sdk-preview

If you install both lusid-sdk-python and lusid-sdk-python-preview, import lusid will refer to lusid-sdk-python.

Configuring API Client

Now that you’ve installed the SDK, it’s time to use it. First, import the following modules:

import lusid
import lusid.models as models
from lusid.utilities import ApiConfigurationLoader

You can create the API factory by running the following code:

secrets_file_path = "/path/to/secrets.json"
config = ApiConfigurationLoader.load(secrets_file_path)
api_factory = lusid.utilities.ApiClientFactory(
    token=lusid.utilities.RefreshingToken(config),
    api_secrets_filename=secrets_file_path
)

You will need to create a secrets file containing your LUSID credentials. For instructions on how to construct this file, see Configure secrets file.

Now that the API client is ready, you are ready to use the various API endpoints. You can list all the API endpoints by running the following:

[api for api in dir(lusid.api) if "API" in api]

An API endpoint can be constructed by calling api_factory.build(lusid.api.<className>) for any of the returned classes. For example, to construct the Transactions Portfolio API, run the following:

tx_portfolios_api = api_factory.build(lusid.api.TransactionPortfoliosApi)
